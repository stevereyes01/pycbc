class pycbc.events.simd_threshold.ThreshClusterObject(series, window, segsize=32768, verbose=0)[source]¶

Bases: object

This class takes a complex SNR time series, a real threshold value, and a window size (expressed as a number of complex sample points), and optionally a segment size and verbosity indicator (for debugging weave).

The execute method returns two numpy arrays: one giving the location of clustered maxima above the threshold, and the other the corresponding (complex) values of the SNR time series at those clustered maxima. The expectation is that the memory of the SNR time series will be filled new inputs many times, and execute() called repeatedly.
